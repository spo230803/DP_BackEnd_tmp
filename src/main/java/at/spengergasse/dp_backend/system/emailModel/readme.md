In diesem Paket finden Sie alle Kurse zum Versenden von E-Mails.
Es gibt einen Dienst, der die Aufgabe hat, E-Mails zu versenden.
Es gibt eine abstrakte Klasse namens "EmailModelBase", die die Basis für alle Modellklassen bildet Ein "Model" ist eine Vorlage, die den Hauptteil der E-Mails enthält

Per ein neues E-Mail-Modell zu erstellen, verwenden Sie die Basisklasse „EmailModelBase“ und implementieren bzw. überschreiben Sie die entsprechenden Funktionen.
